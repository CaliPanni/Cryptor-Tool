
# Cryptor Tool v 1.0

a free tool that allows you to encrypt texts or files with the cl0 technique.
## FAQ
### what is the cl0 technique?

the cl0 technique was developed by CaliPanni, it consists of replacing words in ASCII symbols.

### What file types does this tool read?
all file types, especially text files, such as TXT, and .cl0 files.
## Installation

This tool for now only works with 64-bit Windows, to install it just download the .EXE file in the release and run it.
    
## Usage/Examples

It can be used to send important data over the internet or to servers.
